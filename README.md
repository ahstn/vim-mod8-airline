## vim-mod8-airline
mod8 [vim-airline](https://github.com/vim-airline/vim-airline) companion theme

### Installation
By cloning this repo:
```
$ cd ~/.vim/bundle
$ git clone https://github.com/phazyy/vim-mod8-airline.git
```

Add `let g:airline_theme='mod8'` to your ~/.vimrc
